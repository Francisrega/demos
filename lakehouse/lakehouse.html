Autonomous AI Lakehouse for Open Data and AI

Access Iceberg data in-place
Query Apache Iceberg tables stored in cloud object storage across AWS, Azure, and Google Cloud without copying data. Iceberg provides the open table format while the lakehouse provides performance and intelligence.

In-database AI and analytics
AI capabilities such as Select AI, Vector Search, and machine learning run inside the lakehouse engine using the same security and auditing controls as SQL.
